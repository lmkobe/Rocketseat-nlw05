# Rocketseat NLW 05

## Trilha Node.js

Projeto com o objetivo de aprender Node.js

### Configurações do ambiente

https://github.com/lmkobe/Rocketseat-nlw05.git

### Comandos importantes

yarn init -y

yarn add express

yarn add @types/express -D

yarn add typescript -D

yarn tsc --init

yarn add ts-node-dev -D

yarn dev
